# ENVS-193DS Spring 2026 Final

## General Information

This repository contains the final assignment for ENVS 193DS (Environmental Studies Data Science), Spring 2026. The assignment consists of three problems: reviewing statistical communication about shorebird habitat use in the San Francisco Bay, analyzing the influence of tree height and acacia ant species on bird nest occurrence in an African savanna, and analyzing personal data on caffeinated drink consumption by time of day.

**Author:** Tadeh Peroomian
